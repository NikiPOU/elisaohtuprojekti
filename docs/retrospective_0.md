# Retrospektiivi (Sprint 0)

## Start
- Kertoo tarkemmin muille, mitä taskia tekee esim. discordin kautta
- Jos tekee taskia, task pitäisi lisää github projects:iin ja lisää kaikki taskin tiedot (Duration yms.)
- Pitää muita ryhmäläisiä ajantasalla omasta edistyksestä, että kaikki ymmärtävät tehtyjä asioita -> tarvittaessa esittelee muille
- Etäohjelmointia myös yhdessä -> laittaa discordissa viestiä, jos haluaa työstää yhdessä taskia
- Työnjako tasaisemmaksi, huomioi taskien koot yms. -> todennäköisesti helpottuu sprint 0 jälkeen
- Livetapaamisia pienemmissä ryhmissä halukkaille

## Stop
- Asioiden tekeminen taskien/userstoryen ulkopuolella -> lisää itse backlogiin asiat, joita aikoo tehdä

## Continue
- Kertoo, kun aikoo työntää githubiin jotain
- Hyvää kommunikaatiota
- Halua oppia lisää, omissa brancheissä kokeilua
- Daily Scrumit
